# K9-AIF Website

This repository contains the static website for the K9-AIF framework.

The site provides an overview of the architecture, design philosophy,
and links to the main resources including documentation, examples,
and research material.

## Live Site

https://k9aif.com

## Structure

- `index.html` — Main landing page
- `images/` — Architecture diagrams and assets

## Source

The K9-AIF framework source code is available here:
https://github.com/k9aif/k9-aif-framework

## Notes

This repository is intentionally minimal and serves as a public entry
point to the K9-AIF architecture and ecosystem.
